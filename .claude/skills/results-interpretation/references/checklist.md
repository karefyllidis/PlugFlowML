# Results Interpretation Checklist

Anchoring
- [ ] What was measured, and under what conditions, is restated
- [ ] Units and uncertainty reported
- [ ] Compared against a reference (baseline / target / prior)

Honesty
- [ ] Effect is larger than noise/variance (not over-reading)
- [ ] Statistical vs. practical significance distinguished
- [ ] Single-run results not presented as robust
- [ ] Negative/null results reported, not hidden
- [ ] Numbers shown behind each claim

Reasoning
- [ ] Correlation not claimed as causation
- [ ] In-distribution result not claimed as general
- [ ] Alternative explanations considered
- [ ] Leakage / cherry-picking / survivorship bias checked

Separation
- [ ] Fact ("data shows X") vs. interpretation ("suggests Y") vs. recommendation ("do Z") kept distinct

Limitations & next steps
- [ ] Sample size, confounders, assumptions, generalization limits stated
- [ ] Concrete next steps given
