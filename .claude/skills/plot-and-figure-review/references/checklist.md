# Plot & Figure Review Checklist

Correctness
- [ ] Chart type fits the data
- [ ] Axes show the correct variables; transforms stated
- [ ] No swapped series / wrong aggregation / mislabeled groups

Clarity
- [ ] Both axes labeled with units
- [ ] Readable tick density and font sizes
- [ ] Title/caption lets the figure stand alone
- [ ] Legend present and series distinguishable
- [ ] Colorblind-safe palette; color carries meaning
- [ ] Uncertainty shown (error bars / CI) and defined

Honesty
- [ ] Axis ranges not truncated to exaggerate
- [ ] Bars start at 0 unless justified
- [ ] Linear/log scale chosen to inform, not mislead
- [ ] Dual axes avoided or clearly marked
- [ ] No cherry-picked window
- [ ] Sample size visible where relevant

Fit
- [ ] The figure actually supports the claim it illustrates
